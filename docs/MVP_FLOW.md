# SatyaSetu MVP Flow

1. User receives a suspicious message.
2. User takes a screenshot.
3. User uploads the screenshot to SatyaSetu.
4. OCR extracts text from the image.
5. Sensitive data is redacted (Aadhaar, PAN, Phone, etc.).
6. Scam detection engine analyzes the redacted content.
7. Trust score is generated.
8. User receives an explanation and recommendation.
